# Filtering and Searching Your Tree

Finding people and relationships in large trees is easy with TheModernFamily's search and filter tools.

## Using the Search Bar
- Type a name or keyword to quickly find people or relationships.

## Applying Filters
- Use the sidebar to filter by:
  - Name
  - Date of birth/death
  - Relationship type
  - Status (alive, deceased, etc.)

## Advanced Tips
- Combine filters for more precise results.
- Use the timeline or globe view for time/location-based searches.
