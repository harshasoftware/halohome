# Navigating the Interface

TheModernFamily's interface is designed to be intuitive and powerful. Here's a quick tour:

## Main Areas
- **Family Tree Canvas**: The central area where you build and view your tree.
- **Toolbar**: Quick actions like add person, import/export, undo/redo, and view switcher.
- **Sidebar**: Access filters, search, and project management tools.
- **Modals**: Pop-up dialogs for editing people, relationships, and settings.

## Switching Views
- Use the view switcher to toggle between:
  - **Tree View**: Standard family tree layout
  - **Timeline View**: See events and relationships over time
  - **Globe View**: Visualize migrations and locations

## Tips
- Right-click on nodes or edges for context menus.
- Use zoom and pan to navigate large trees.
- The search bar helps you quickly find people or relationships.
